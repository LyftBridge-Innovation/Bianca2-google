Here's the complete Lyftbridge color system:

Core Brand Colors
Color	Hex Code	What It Is
Lyftbridge Blue	#296CAC	The primary brand color — used everywhere as the main accent
Lyftbridge Black	#000000	Used for title slide backgrounds, body text on light backgrounds
Lyftbridge White	#FFFFFF	Text on dark backgrounds, card backgrounds
Supporting "Casamassima Blue" Family
These are three complementary blues that create depth and visual hierarchy:

Color	Hex Code	Role
Casamassima Blue 1	#5D86AD	Secondary accent — supporting visuals, lighter elements, footer text
Casamassima Blue 2	#93B1D4	Soft accent — subtle backgrounds, dividers, borders
Casamassima Blue 3	#14729B	Deep accent — emphasis, call-to-action elements, high-contrast elements
How They're Used Across Documents
Presentations (PPTX): Title slides get black backgrounds with white text. Content slides use light gray (#F5F7FA) or white backgrounds. The blues are used for shape fills, headers, accent bars, and gradients. Logo on the title slide, copyright footer on every slide.

Spreadsheets (XLSX): Header rows get Lyftbridge Blue background with white text. Alternating rows use a light blue tint (#E8EFF7). Subtotal rows use Casamassima Blue 1. Borders use Casamassima Blue 2.

Word Documents (DOCX): Headings in Lyftbridge Blue. Body text in black. Accent borders and highlights in Casamassima Blue 3. Table headers use the light blue tint. Footers in Casamassima Blue 1.

Dashboard UI Colors
The dashboard itself uses the same family translated to HSL:

Primary: hsl(210, 85%, 45%) — maps closely to Lyftbridge Blue
Secondary: hsl(210, 70%, 55%) — lighter variant
Background: Dark theme at hsl(220, 20%, 8%) — near-black with a slight blue tint
Cards/panels: hsl(220, 18%, 12%) — slightly lighter dark blue
Muted text: hsl(215, 15%, 60%) — soft gray-blue
Charts: Five shades stepping from the primary blue through teals
Everything ties back to the same blue family — it's a cohesive system that flows from the dashboard all the way through to the AI-generated documents.
